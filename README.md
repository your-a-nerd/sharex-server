# ShareX Server

# :floppy_disk: Installation

```bash
# clone the repo
$ git clone https://github.com/greatplainsmodding/sharex-server

$ cd sharex-server

# install the node modules...
$ npm install

# start
$ npm run build
$ npm start
```

Note this is still in development so it will not work.
